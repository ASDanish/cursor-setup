# Cursor IDE Setup

This repository documents setting up Cursor IDE with the Claude Code and Codex extensions.

## Tools Installed

- **Cursor IDE** — AI-powered code editor (cursor.com)
- **Claude Code for VS Code** — extension by Anthropic
- **Codex** — coding agent extension by OpenAI

## Steps Completed

1. Installed Cursor IDE from cursor.com and signed in.
2. Installed the Claude Code extension via Extensions and logged in with my Anthropic account.
3. Installed the Codex extension via Extensions and logged in with my OpenAI account.
4. Created this public GitHub repository (cursor-setup).
5. Cloned the repository and opened it in Cursor.
6. Wrote this README documenting the process.
7. Committed and pushed the changes to GitHub.

## Issues & Solutions

- **Couldn't find the Extensions panel at first.** Cursor opened in the Agent Window, which doesn't show the editor sidebar. Switched to the Editor Window, then opened Extensions via View → Extensions.
- **Both extensions were already installed but not logged in.** Used the Command Palette (Ctrl+Shift+P) to open each extension's sidebar and signed in to both.
